a leaderboard for models trying to design long-lived starting boards in Conway's Game of Life

about the benchmark

Each model is asked to propose an initial board on an {{ rows }}x{{ cols }} toroidal grid.

The rule system is Conway's Game of Life.

We run each model across {{ trials_per_model }} trial seeds and keep its best submission.

The main score is the number of steps before the first repeated global state, capped at {{ max_steps }} steps.
